# py-websocket-online-game

WebSocketを使ってオンラインゲームを開発するためのリポジトリです。

## 構成

- クライアント：GitHub Pages
- サーバー：Render（Python WebSocketサーバー）

## 技術
- Python（websockets）
- JavaScript（WebSocket API）
- GitHub Pages（フロントエンド）
- Render（バックエンド）
